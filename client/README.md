# Butcher Order System (Full-Stack)

Full-stack web application for managing butcher shop orders in a real business environment.

## Tech Stack
Frontend
- React
- Vite

Backend
- Node.js
- Express

Database
- SQLite

## Features
- Create new orders
- Manage different order types (lamb, regular, etc.)
- Price calculation
- Order history
- Future orders management
- Archive system

## Project Structure

client/
React frontend

server/
Node.js + Express backend API

## Run Locally

Backend
npm install
node src/index.js

Frontend
npm install
npm run dev
