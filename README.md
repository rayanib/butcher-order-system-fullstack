# Butcher Order System (Full-Stack)

A full-stack web application for managing butcher shop orders and workflow.

This project was built to support real-world operations in a butcher shop, including creating orders, tracking different order types, calculating prices, and organizing order history.

The system replaces manual order tracking and helps manage daily shop operations more efficiently.

---

## Tech Stack

Frontend
- React
- Vite
- Context API

Backend
- Node.js
- Express.js

Database
- SQLite (or local storage depending on configuration)

---

## Features

- Create and manage new orders
- Track different order types (lamb, regular, etc.)
- Automatic price calculation
- View order history
- Archive completed orders
- Manage future orders
- Clean and simple interface designed for real shop workflow

client/

## How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/rayanib/butcher-order-system-fullstack.git
```

---

### 2. Run the Backend

```
cd server
npm install
node src/index.js
```

Backend server will run on:

```
http://localhost:3000
```

---

### 3. Run the Frontend

Open another terminal and run:

```
cd client
npm install
npm run dev
```

Then open:

```
http://localhost:5173
```
