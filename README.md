# Butcher Order System (Full-Stack)

A full-stack web application for managing butcher shop orders and workflow.

This project was built to handle real-world operations in a butcher shop, including creating orders, tracking items, and organizing order history.

## Tech Stack

Frontend
- React
- Vite

Backend
- Node.js
- Express

## Features

- Create and manage new orders
- Track different order types (lamb, regular, etc.)
- Price calculation
- Order history and archives
- Future order planning
